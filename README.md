# Online shop

# Getting Starting
Change credentials in config/database.yml to another

```
username: username
password: Strong_Password_#!1
host: localhost
port: 5433
```
```
> rake db:create
> rake db:migrate
```
```
rails s -p 7000
```
Open browser with url http://localhost:7000/
Have fun :)
